SSN (Social Security Number)

Formatted SSN

Social Security Numbers (SSNs) are typically formatted with specific separators between groups of digits. Valid examples include 123-45-6789, 234 56 7890, and 345.67.8901. These formats use hyphens, spaces, or periods to separate the three groups of digits, making them easily recognizable as SSNs. Another acceptable format uses underscores, as in 456_78_9012.

Other valid examples of formatted SSNs include 567-89-0123, 678 90 1234, 789.01.2345, and 890_12_3456. It's important to note that valid SSNs can start with any digit, including zero, as demonstrated by 001-23-4567 and 112 34 5678. The key is maintaining the correct grouping of digits (3-2-4) regardless of the separator used.

Invalid SSNs often violate specific rules set by the Social Security Administration. For instance, 666-12-3456 and 000-12-3456 are invalid because SSNs don't start with 666 or 000. Similarly, 901-23-4567 is invalid as SSNs don't begin with 900 series numbers. Numbers with all zeros in any group, like 123-00-4567 or 234-56-0000, are also invalid.

Other examples of invalid formatted SSNs include those with incorrect number of digits, such as 345-67-89012 (too many digits) or 456-78-901 (too few digits). Improper formatting, like 56789-0123 or 678.901234, also renders an SSN invalid. Lastly, SSNs containing non-numeric characters, such as 789-01-234A, are not valid.

Unformatted SSN

Unformatted Social Security Numbers appear as a continuous string of nine digits without any separators. Valid examples include 123456789, 234567890, and 345678901. These numbers, while lacking the typical hyphens or spaces, still represent legitimate SSNs. Other valid unformatted SSNs are 456789012, 567890123, and 678901234.

Additional instances of valid unformatted SSNs include 789012345, 890123456, 001234567, and 112345678. As with formatted SSNs, unformatted numbers can start with any digit, including zero. The key is that they contain exactly nine digits in the correct sequence, even without visible grouping.

Similarly, 901234567 is invalid as SSNs don't begin with 900 series numbers. Numbers with all zeros in any group, like 123004567 or 234560000, are also invalid.

SSNs containing non-numeric characters, like 567890123A, are not valid.
